# Reference geometry

Drop reference STEP files here. They're imported by `src/viz.py` and
appear in `assembly.step` for dimensional verification — they're never
exported as printable parts.

Expected files:

| File                          | Source                                                                 | Purpose                                  |
|-------------------------------|------------------------------------------------------------------------|------------------------------------------|
| `makita_battery.step`         | [GrabCAD — Makita LxT 18V 5A Battery](https://grabcad.com/library/makita-lxt-18v-5a-battery-1) | Battery dock interface reference         |

Files in this directory are not committed (large binary; license varies
by source). Each contributor downloads them from the linked source.
